🚀 Modern JavaScript (ES6+)

Modern JavaScript introduced powerful new syntax that makes code cleaner and more readable. These features are highly utilized in React.

1. Destructuring

Destructuring allows you to unpack values from arrays or properties from objects into distinct variables.

Object and Array Destructuring
// Object Destructuring
const user = { name: 'Bob', age: 25, role: 'Admin' };
const { name, role } = user;
console.log(name, role); // "Bob" "Admin"

// Array Destructuring
const rgb = [255, 128, 0];
const [red, green, blue] = rgb;
console.log(green); // 128

2. Spread and Rest Operators (...)

💡 React Prerequisite

The Spread operator is used constantly in React for copying and updating State without mutating the original object.

Spread Operator
// Copying Arrays
const arr1 = [1, 2];
const arr2 = [...arr1, 3, 4]; // [1, 2, 3, 4]

// Copying and Updating Objects
const person = { name: 'Sarah', age: 28 };
const updatedPerson = { ...person, age: 29, location: 'NY' };
// { name: 'Sarah', age: 29, location: 'NY' }

3. ES Modules (Import/Export)

Modules allow you to split your code into multiple files.

math.js (Exporting)
// Named export
export const PI = 3.14159;

// Default export (only one per file)
export default function add(a, b) {
  return a + b;
}
app.js (Importing)
// Import default and named exports
import add, { PI } from './math.js';

console.log(PI); // 3.14159
console.log(add(2, 3)); // 5
